For the 2024 slots, TCS favored "Array Transformation" and "Number Theory" problems.
Problem: The Chocolate Factory (Most Repeated)Description: Move all zeros (empty packets) to the end of the array while keeping the relative order of non-zero elements.Complexity: Required $O(n)$ time and $O(1)$ space.
Problem: Toggling Bits (Digital Logic)Description: Take an integer, convert it to binary, flip all bits, and return the new decimal value.
Problem: Airport Security (Risk Severity)Description: Sort an array of $[0, 1, 2]$ representing risk levels in a single pass (Dutch National Flag algorithm).
